# Zichao Xia — Academic Homepage

Personal academic website built with [Hugo](https://gohugo.io/) and the
[Minimal Academic](https://github.com/jhu247/minimal-academic) theme.

## Local development

Install Hugo (extended), then run:

```bash
hugo server
```

Open http://localhost:1313/ in your browser.

## Deployment

The site is automatically built and deployed to GitHub Pages via the workflow in
`.github/workflows/hugo.yml` on every push to the `main` branch.
